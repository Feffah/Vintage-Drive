# Vintage Cars E-commerce (React + Vite)

This project is a React + Vite e-commerce application for Vintage Cars, using JavaScript and JWT authentication.

## Features
- User authentication (JWT-based login & registration)
- Product listing for vintage cars
- Shopping cart management
- Modern, responsive UI

## Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Start the development server:**
   ```sh
   npm run dev
   ```

## Project Structure
- `src/` — React components, pages, and assets
- `public/` — Static files
